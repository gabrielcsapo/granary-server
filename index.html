What is Granary?


Granary is an NPM / Bower bundling Server

problem
It does not replace NPM or bower when it comes to depency maintance. Granary is a project bundler. When developing Node.js applications NPM is widely used for package depencies. Depending on the speed of your internet NPM might take quite a while to multiplex your downloads and compile them to finally begin development.

solution

Granary helps by being deployed on cloud provider and access to high speed internet it will bundle your npm and bower dependencies and send them to you as one gzipped folder which it will unpack into the corresponding node_modules and bower_components directories ready to use.

Install Granary


npm install granary -g

What Does Granary Do?

Granary can create production and development bundles of your projects
granary create -u=http://localhost:8872
Granary installs the development bundle by default
granary -u=http://localhost:8872
Granary also allows you to specify production bundle 
granary -u=http://localhost:8872 --production

Install your own Server!

Clone the server
git clone https://github.com/gabrielcsapo/granary-server.git
Install a redis server
brew
brew install redis
apt 
sudo apt-get -y install redis-server;
Install node dependencies 
npm install
Start the server!
npm start;

Start the server with a custom config

GRANARY_CONFIG=/ npm start;
